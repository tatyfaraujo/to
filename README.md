## Como usar

### Localmente
1. Abra o arquivo `index.html` em um navegador
2. Ou use um servidor local:
   ```bash
   python -m http.server 8000
   # ou
   npx http-server
   ```

### No GitHub Pages
1. Crie um repositório chamado `tatyanny-portfolio`
2. Faça upload de todos os arquivos (incluindo as pastas `data/` e `images/`)
3. Vá em Settings → Pages
4. Selecione "Deploy from a branch"
5. Escolha a branch "main" e pasta "/ (root)"
6. Clique em Save

Seu site estará disponível em: `https://tatyfaraujo.github.io/tatyanny-portfolio/`
